## 배열
- 중복이라는 키워드가 나오면 `Set`을 활용해보자!
- 중간에 데이터를 삽입할 일이 많다면 배열은 비효율적이다. 다른 방법을 생각해보자.
- 많이 사용했던 함수: `enumerated()`, `indicies`, `filter`, `map`, `compactMap`, `sorted`, `reversed` `abs`
